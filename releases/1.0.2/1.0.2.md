# OmegaGUI 1.0.2 - 2025-11-02

## Added
- Added a way to track private accounts locally only: right-click an account to mark it as private; it will display the suffix "(private)".
- The program will now prompt to update automatically.

## Notes
- Privacy markers are stored locally and are not synced to any cloud service.
- This release focuses on usability improvements for account management and update notifications.
