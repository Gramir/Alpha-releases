# Version 1.0.0 - Initial Release with Auto-Updater

## Features
- Game token management system
- Steam account integration  
- ZIP file export functionality
- Modern WPF interface
- **NEW: Auto-updater integration for seamless updates**

## Improvements
- Enhanced UI/UX with dark theme support
- Optimized performance and memory usage
- Improved error handling and logging

## Compatibility
- Windows 7 SP1 and later
- .NET 8.0 Runtime
- Wine 8.0+ support for Linux users

## Installation
Download and run omegaGUI.exe - no installation required.
Updates will be delivered automatically.
